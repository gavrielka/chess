# Grandmaster Chess | Premium Edition

A fully functional, beautiful chess game in a single HTML file. Built with vanilla JavaScript, CSS, and HTML5.

## ✨ Features

- **Full Chess Rules**: Castling, En Passant, Pawn Promotion, and Check/Checkmate/Stalemate detection.
- **AI Opponent**: Play against a Minimax AI with Alpha-Beta Pruning (Settings for Intro, Challenge, and Grandmaster).
- **Two-Player Mode**: Local hot-seat play for two humans.
- **Move Visualization**: Click-to-select and move with legal move highlighting and capture indicators.
- **Move History**: Complete record of the game in standard algebraic notation.
- **Robust Undo**: Reverse any number of moves at any time.
- **Premium Design**: Dark, luxury aesthetic with custom CSS-textured board and responsive layout.
- **Audio Feedback**: Synthesized sound effects using the Web Audio API (Move, Capture, Check).
- **Game Status**: Real-time turn indicators and "check" alerts.

## 🚀 How to Run

Simply open [chess.html](chess.html) in any modern web browser. No dependencies or installation required.

## 🎮 Controls

- **Select Piece**: Left-click on any of your pieces.
- **Move**: Click on a highlighted legal move (Green dot for normal move, Red square for capture).
- **Mode Toggle**: Click the "Mode" button to switch between **Human vs Human** and **vs Computer**.
- **Difficulty**: Use the dropdown to set the AI level.
- **Undo**: Use the **Undo Move** button to step back.

## 🛠️ Technical Stack

- **Core**: HTML5, Vanilla JavaScript.
- **Styling**: CSS3 (with custom gradients for textures).
- **AI**: Minimax algorithm with Alpha-Beta pruning (Depth 3).
- **Audio**: Web Audio API (Synthesized oscillators).
